## 1. 产品形态与分发方式

你现在做的是 **Web 应用**（`index.html`），有以下几种分发方式：

### 方式对比

| 方式 | 说明 | 优点 | 缺点 | 推荐度 |
|------|------|------|------|--------|
| **静态托管** | Vercel/Netlify/GitHub Pages | 免费、简单、自动 HTTPS | 需要域名 | ⭐⭐⭐⭐⭐ |
| **云服务器** | 阿里云/腾讯云 VPS | 完全控制、可定制 | 需要运维、有成本 | ⭐⭐⭐⭐ |
| **对象存储** | 阿里云 OSS/腾讯云 COS | 便宜、稳定 | 需要配置 CDN | ⭐⭐⭐⭐ |
| **Electron 打包** | 做成桌面应用 | 可离线使用 | 容易被破解 | ⭐⭐⭐ |
| **直接发 HTML** | 发给用户本地打开 | 最简单 | License 验证会被绕过 | ❌ 不推荐 |

### 推荐方案：Vercel 静态托管

**为什么推荐**：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN
- ✅ 自动部署（连接 GitHub）
- ✅ 支持自定义域名

**操作步骤**：
1. 访问 https://vercel.com/
2. 用 GitHub 账号登录
3. 导入你的 GitHub 仓库
4. 自动部署，获得网址：`https://your-project.vercel.app`

## 3. 支付接入方案

### 方案对比

| 支付方式 | 接入难度 | 费率 | 结算周期 | 适用场景 |
|---------|---------|------|---------|---------|
| **微信支付官方** | ⭐⭐⭐⭐ | 0.6% | T+1 | 有营业执照的企业 |
| **支付宝官方** | ⭐⭐⭐⭐ | 0.6% | T+1 | 有营业执照的企业 |
| **Stripe** | ⭐⭐ | 2.9% + $0.3 | 7 天 | 国际用户、有英文站 |
| **爱贝云支付** | ⭐⭐ | 3-5% | T+1 | 个人开发者、国内用户 |
| **虎皮椒支付** | ⭐⭐ | 3-5% | T+1 | 个人开发者、国内用户 |
| **手动收款** | ⭐ | 0 | 即时 | 初期测试、小额交易 |

### 方案一：手动收款（推荐初期使用）

**适合场景**：刚上线、日订单<50、测试阶段

**优点**：
- ✅ 零成本
- ✅ 无需技术接入
- ✅ 即时到账
- ✅ 可以和用户直接沟通

**缺点**：
- ❌ 需要人工操作
- ❌ 无法 24 小时自动发货
- ❌ 用户信任度低

**操作步骤**：
1. 在付费弹窗显示微信/支付宝收款二维码
2. 用户扫码转账（备注邮箱）
3. 你收到钱后，在管理后台生成激活码
4. 手动发送邮件给用户

**代码修改**：
```javascript
// 在 selectPayment 函数中，显示你的收款码
if (method === 'wechat') {
    qrContent = `
        <div style="padding:20px;text-align:center">
            <img src="你的微信收款码图片 URL" style="width:200px;height:200px">
            <p style="margin-top:15px;font-size:14px">
                💚 微信扫码支付 ¥39（终身版）
            </p>
            <p style="font-size:12px;color:#888">
                支付后请保留截图，并添加微信：yourwechat 发送截图获取激活码
            </p>
        </div>
    `;
}
```

### 方案二：第三方支付（推荐中期使用）

**适合场景**：日订单>20、需要自动发货

**推荐平台**：

| 平台 | 官网 | 特点 |
|------|------|------|
| 爱贝云 | https://www.ibeecloud.com/ | 支持微信/支付宝，个人可用 |
| 虎皮椒 | https://www.xunhupay.com/ | 接口简单，文档完善 |

**接入步骤**（以爱贝云为例）：
1. 注册账号，完成实名认证
2. 创建应用，获取 APP_ID 和 APP_SECRET
3. 在服务器部署支付回调代码
4. 前端调用支付接口
5. 用户支付成功后，自动发货

**示例代码**：
```javascript
// 调用爱贝云支付
async function createOrder(amount, email) {
    const response = await fetch('https://api.ibeecloud.com/2.0/application/prepare', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            appId: '你的 APP_ID',
            money: amount * 100, // 单位：分
            source: 'wechat', // 或 alipay
            sourceReturnUrl: 'https://yourdomain.com/return',
            sourceNotifyUrl: 'https://yourdomain.com/notify',
            state: email // 传递用户邮箱
        })
    });
    const data = await response.json();
    // 跳转到支付页面
    window.location.href = data.data.qrCodeUrl;
}
```

### 方案三：官方支付（推荐长期发展）

**适合场景**：有营业执照、日订单>100

**微信支付接入**：
1. 注册微信支付商户（需要营业执照）
2. 下载微信支付 SDK
3. 部署服务器代码
4. 前端调用微信支付 API

**支付宝接入**：
1. 注册支付宝开放平台（需要营业执照）
2. 创建应用，获取 APP_ID
3. 下载支付宝 SDK
4. 部署服务器代码

## 4. 广告接入方案

### 广告平台对比

| 平台 | 适用场景 | CPM 估算 | 结算方式 | 推荐度 |
|------|---------|---------|---------|--------|
| **Google AdSense** | 有国际用户 | $1-5 | 美元，$100 起付 | ⭐⭐⭐⭐ |
| **百度联盟** | 国内用户 | ¥2-10 | 人民币，¥100 起付 | ⭐⭐⭐⭐ |
| **腾讯广告** | 微信生态 | ¥1-8 | 人民币 | ⭐⭐⭐ |
| **穿山甲** | 移动端 App | ¥3-15 | 人民币 | ⭐⭐⭐ |
| **Google AdMob** | 移动 App | $2-8 | 美元 | ⭐⭐⭐⭐ |

### 方案一：Google AdSense（推荐）

**适合场景**：有英文内容或国际用户

**接入步骤**：
1. 访问 https://adsense.google.com/
2. 用 Google 账号登录
3. 添加网站（需要已上线的网站）
4. 添加网站验证代码到 `<head>`
5. 等待审核（1-7 天）
6. 审核通过后，创建广告单元
7. 复制广告代码到网站

**代码示例**：
```html
<!-- 在你的广告位容器中添加 -->
<div id="adContainer">
    <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
    <ins class="adsbygoogle"
         style="display:block"
         data-ad-client="ca-pub-xxxxxxxxxxxx"
         data-ad-slot="1234567890"
         data-ad-format="auto"></ins>
    <script>
         (adsbygoogle = window.adsbygoogle || []).push({});
    </script>
</div>
```

**替换位置**：`index.html` 中的 `renderSidebarAd()` 函数

### 方案二：百度联盟（推荐）

**适合场景**：纯中文用户

**接入步骤**：
1. 访问 https://union.baidu.com/
2. 注册账号，实名认证
3. 添加网站，等待审核
4. 审核通过后，创建广告位
5. 复制代码到网站

### 当前代码中的广告位

你的代码中已有广告位占位符：

```javascript
// 在侧边栏底部添加广告位
function renderSidebarAd() {
    const container = document.getElementById('sidebarAdContainer');
    if (!container) return;

    if (checkPremiumStatus()) {
        // Premium 用户：清空广告
        container.innerHTML = '';
        return;
    }

    // 非 Premium 用户：显示广告
    container.innerHTML = `
        <div class="ad-banner">
            <button class="ad-close" onclick="this.parentElement.style.display='none'">×</button>
            <div class="ad-banner-title">广告 · Ad</div>
            <div class="ad-banner-content">
                🎯 想要去除广告？
                <button onclick="showPremiumModal()">👑 升级专业版</button>
            </div>
        </div>
    `;
}
```

**替换为真实广告**时，保留 Premium 用户判断逻辑，只替换广告内容部分。

## 5. 部署上线流程

### 步骤一：准备代码

1. 确保代码在 Git 仓库中
2. 修改 Supabase 配置（`index.html` 和 `admin.html`）
3. 测试本地功能正常

### 步骤二：部署到 Vercel

1. 访问 https://vercel.com/
2. 用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择你的 GitHub 仓库
5. 点击 "Deploy"
6. 等待部署完成（约 1 分钟）
7. 获得网址：`https://your-project.vercel.app`

### 步骤三：配置自定义域名（可选）

1. 购买域名（阿里云/腾讯云/Namecheap）
2. 在 Vercel 设置中添加自定义域名
3. 按照提示配置 DNS
4. 等待 DNS 生效（约 30 分钟）

### 步骤四：配置环境变量

如果使用了敏感配置，建议使用 Vercel 环境变量：

1. Vercel 项目设置 → Environment Variables
2. 添加：
   - `SUPABASE_URL`
   - `SUPABASE_KEY`
   - `RESEND_API_KEY`（如果使用邮件功能）

### 步骤五：测试上线

1. 访问部署后的网址
2. 测试所有功能
3. 测试付费流程（可以用 1 分钱测试）
4. 检查邮件发送是否正常

## 附录 B：常见问题

### Q1: 个人可以接入微信支付吗？
A: 不能。微信支付需要营业执照。个人可以使用第三方支付平台（爱贝云、虎皮椒）或手动收款。

### Q2: 广告联盟好申请吗？
A: AdSense 需要网站有一定内容且能访问；百度联盟需要实名认证和网站审核。建议先上线网站，有内容后再申请。

### Q3: 如何防止激活码被破解？
A:
1. 使用云端验证（已实现）
2. 设备绑定（已实现）
3. 定期更换激活码格式
4. 监控异常激活行为

### Q4: 需要 ICP 备案吗？
A: 如果服务器在中国大陆且面向国内用户，需要备案。使用 Vercel 等海外服务暂不需要。

### Q5: 如何开发票？
A: 个人无法开具增值税发票。可以：
1. 初期：不开发票，在页面说明
2. 中期：找第三方代开发票
3. 长期：注册公司，申请开票资质
